<h3>오픈소스 OS, Linux</h3>
Linux는 여러 유저들이 공동으로 제작해서 만드는 오픈소스 OS이다.  
Linux는 1991년에 처음 개발되었다. 
Linux가 많은 사랑을 받는 이유는 다음과 같다.
  
1. 폭넓은 하드웨어 지원  
2. 안정성, 안전성  
3. 가격(무료)  
4. 보안성  
5. 사용자 자율성  
  
누구든지 컴퓨터에 조금만 관심이 있다면 리눅스를 한번 사용해 보면 좋겠다.
